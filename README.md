# 🇹🇭 Thai Alphabet Block

[![Minecraft 1.20.1](https://img.shields.io/badge/Minecraft-1.20.1-brightgreen.svg)](#)
[![Fabric](https://img.shields.io/badge/Fabric-Supported-orange.svg)](#)
[![Forge](https://img.shields.io/badge/Forge-Supported-red.svg)](#)

A Minecraft mod that adds beautiful Thai alphabet blocks to the game! Perfect for building signs, decorating schools, or creating Thai-themed maps. This mod is built for **both Forge and Fabric** on Minecraft 1.20.1.

---

## ✨ Features

- **69 Thai Character Blocks**: Includes all major consonants (ก-ฮ), vowels, and tone marks as placeable full blocks.
- **Customizable Colors (Dyeable!)**: Right-click any Thai alphabet block with a Dye to change its background color! The wood-like background will beautifully change its shade while keeping the character clearly visible.
- **Natural Wood Texture**: By default, blocks feature a natural warm wood panel texture that blends perfectly with vanilla building blocks.
- **Dedicated Creative Tab**: Easily find all 69 characters organized in their own custom creative tab.
- **Multi-Loader Support**: Works seamlessly on both Fabric and Forge (1.20.1).

## 🎨 How to Dye Blocks

1. Place any Thai Alphabet Block.
2. Hold any of the 16 Minecraft Dyes in your hand.
3. **Right-Click** the block! The background will instantly change to the dye's color.

## 📥 Installation

1. Make sure you have installed [Fabric Loader](https://fabricmc.net/) or [Minecraft Forge](https://files.minecraftforge.net/) for `1.20.1`.
2. _(For Fabric)_ Install the [Fabric API](https://modrinth.com/mod/fabric-api).
3. Download the mod `.jar` file from Modrinth or the GitHub Releases page.
4. Place the `.jar` file in your `.minecraft/mods` folder.
5. Launch the game and enjoy!

# 🇹🇭 ภาษาไทย

ม็อด Minecraft ที่เพิ่มบล็อกตัวอักษรภาษาไทยเข้ามาในเกม! เหมาะสำหรับนำไปสร้างป้ายข้อความ ตกแต่งอาคาร โรงเรียน หรือสร้างแมพธีมไทยๆ ม็อดนี้รองรับทั้ง **Forge และ Fabric** บนเวอร์ชัน 1.20.1

## ✨ ฟีเจอร์หลัก

- **บล็อกอักษรไทย 69 บล็อก**: ครอบคลุมทั้งพยัญชนะ (ก-ฮ) สระ และวรรณยุกต์
- **ย้อมสีได้ตามใจชอบ (Dyeable!)**: สามารถนำสีย้อม (Dye) ทั้ง 16 สีในเกมมาคลิกขวาที่บล็อกเพื่อเปลี่ยนสีพื้นหลังของบล็อกได้เลย! โดยลวดลายพื้นหลังจะเปลี่ยนสีไปตาม Dye แต่ตัวอักษรจะยังคงเป็นสีน้ำตาลเข้มอ่านง่ายเหมือนเดิม
- **ลวดลายไม้ธรรมชาติ**: สีเริ่มต้นของบล็อกจะเป็นลายแผ่นไม้ธรรมชาติที่เข้ากับบล็อกไม้ต่างๆ ในเกมได้เป็นอย่างดี
- **หน้าต่าง Creative Tab แยกเฉพาะ**: หาง่าย ไม่ปะปนกับของอื่นๆ เพราะมีหน้าต่างช่องเก็บของแยกเฉพาะสำหรับตัวอักษรไทยทั้งหมด
- **รองรับ Multi-Loader**: สามารถเล่นได้ทั้งบน Fabric และ Forge (1.20.1)

## 🎨 วิธีย้อมสีบล็อก

1. วางบล็อกตัวอักษรไทยที่ต้องการ
2. ถือสีย้อม (Dye) สีใดก็ได้ในเกมไว้ในมือ
3. **คลิกขวา** ที่ตัวบล็อก! สีพื้นหลังของบล็อกจะเปลี่ยนเป็นสีนั้นทันที

## 📥 วิธีติดตั้ง

1. ติดตั้ง [Fabric Loader](https://fabricmc.net/) หรือ [Minecraft Forge](https://files.minecraftforge.net/) สำหรับเวอร์ชัน `1.20.1`
2. _(สำหรับสาย Fabric)_ ต้องลงม็อด [Fabric API](https://modrinth.com/mod/fabric-api) ด้วย
3. ดาวน์โหลดไฟล์ `.jar` ของม็อดจาก Modrinth หรือ GitHub Releases
4. นำไฟล์ `.jar` ไปใส่ในโฟลเดอร์ `mods` ของเกม
5. เข้าเกมแล้วสนุกได้เลย!
